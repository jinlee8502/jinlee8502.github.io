Boolean Operators

Boolean operators that we will need in this course are “and”, “or”, and “not”.

And: It evaluates to “True” if the left side AND the right side are “True” in value. If at least one of them is “False”, the whole thing is “False”.

Or: It evaluates to “True” even if one of them is “True”. Only if both arguments are “False”, it evaluates to “False”.

Not: It just makes “True” “False” and makes “False” “True”.

The built-in comparison operations are >, <, >=, <=, ==, and !=.

>: "greater than"

>=: "greater than or equal to"

==: "is equal to"

!=: "is not equal to"

Conditional Statements

A conditional statement in Python consists of a series of headers and suites: a required if clause, an optional sequence of elif clauses, and finally an optional else clause:

if "expression": ...... "suite" elif "expression": ...... "suite" else: ...... "suite"

The content of a "suite" must be indented.

If the expression is "True", the code enters the suite for that clause. Once the suite finishes, the program skips the rest of the clauses. If expression is "False", it moves onto the next cluse.

Loops

Let's go through this function together! A variable named "number" is now 1. A variable named "total" is now 0. And after that, we have something called a loop. Loop is a statement that allows for repetition of a certain function. This "sum()" function has a "while loop".

whlie "expression": ...... "suite"

And this is how you read this while loop.

"While the condition("Number" is less than or equal to 100) is true, repeat this action in the suite: the new "total" becomes the old "total" plus "number". And the new "number" becomes the old "number" plus 1."

Another way of saying that is "keep "incrementing," or adding to, our total by 1, 2, 3, 4 ("number") until "number" reaches 101." The program is essentially adding up all the numbers from 1 to 100 and printing that "total" at the end of the program.
