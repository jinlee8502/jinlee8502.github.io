Flickr Perf Map

I created and maintained Flickr Perf Map, an internal service-engineering dashboard on the Splunk platform that reports global performace statistics in real time. I Utilized Splunk's Search Processing Language, which resembles SQL, to visualize geographic data in 30-minute intervals.

Flickr Health Dashboard

Drawing has haunted me ever since that C- in 4th grade art, but if you let me use my iPad, I'm not too bad. I designed the user interface of Flickr Health Dashboard, an internal mobile application that displays real-time performance statistics for endpoint clients and Flickr infrastructures.

Flickr Error Workflow Dashboard

I developed Error Workflow Dashboard with Splunk’s Machine Learning Toolkit, which detects outliers in the number of internal server errors and captures a snapshot of the errors with various statistics like signed-in user and requested URL proportions

Flickr Time-to-load-Photo-Only Dashboard

I also built Time-to-load-Photo-Only Dashboard, also on Splunk. The application incorporated a search bar that allowed easy lookup of load times grouped by continents and different colocation center origins, with user-selected sample ratio.
